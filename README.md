# 러닝 코스 공유 프로그램 RunShare
![RunShare 배포 다이어그램 - ALB 추가.jpg](assets/RunShare%20%EB%B0%B0%ED%8F%AC%20%EB%8B%A4%EC%9D%B4%EC%96%B4%EA%B7%B8%EB%9E%A8%20-%20ALB%20%EC%B6%94%EA%B0%80.jpg)

## 주요 기능
* 실시간 GPS 위치 정보를 활용한 러닝 실력 측정
* GPX 파일 분석 및 저장
* 가까운 거리순으로 사용자에게 적합한 코스 추천

## 기술 스택
* SpringBoot, JPA
* Docker
* Github Actions
* SupaBase
* AWS S3, AWS ALB, AWS Auto Scaling Group, AWS Route53
